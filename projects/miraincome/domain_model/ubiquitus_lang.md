## ミラインカム　ユビキタス言語

###

| 和名 | 英名 | 説明 | 備考 |
|------|-----|-----|-----|
| 支援企業 | Enterprise | 家庭に支援する企業 | |
| 代表者 | Representative | 支援企業の代表者 | |
| 支援可能家庭数 | SupportedFamilyCount | 支援企業が支援可能な家庭数 | |
| 受給家庭 | Family | 支援を受ける家庭 | |
| カード | CreditCard | クレジットカード| |
| 確認書類 | VerificationDocuments | 支援企業として提出する書類。登記簿謄本、住民票など | |
| 支援金振込先 | FamilyBankAccount | 家庭が支援金を受け取る銀行口座 | |
| 入金履歴 | PaymentHistory | 支援金の入金履歴 | |
| 収入証明書 | IncomeCertificate | 収入証明書。源泉徴収票、納税通知書、住民税決定通知書、給与明細書、所得証明書、確定申告書のいずれか | |
| クエスト | Quest | 支援企業からのタスク。クエストを実行し、レポートを提出すると支援金が支給される | |
| 振込通知 | TransferNotification | 支援金の振込通知 | |
